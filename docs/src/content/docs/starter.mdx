### Installation

<Steps>
1. Ensure you have downloader loaded.

    ```
    [p]load downloader
    ```

2. Add `Seina-Cogs` to your Red instance.

    ```
    [p]repo add Seina-Cogs https://github.com/japandotorg/Seina-Cogs
    ```

3. Install cog(s) from `Seina-Cogs` in your Red instance.

    ```
    [p]cog install Seina-Cogs <cogname>
    ```

4. Load the cog(s)

    ```
    [p]load cogname
    ```
</Steps>
